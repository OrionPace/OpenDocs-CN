# Antigravity 中文文档入口

这里是 Google Antigravity 相关资料的中文整理入口。

## 当前原则

1. **先核验，后翻译**：产品状态、功能边界、许可和价格等信息必须尽量回链官方来源。
2. **不伪装官方**：本站只做社区中文整理，不使用“官方中文”表述。
3. **不混入 Codex**：本项目聚焦 Antigravity；OpenAI Codex 如需翻译，应独立建项。
4. **保留历史，但不背负历史**：Gemini CLI 相关内容仅作为迁移背景和归档材料，不再作为主维护方向。
